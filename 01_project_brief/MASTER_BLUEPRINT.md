# Master Blueprint — DeenTheGreat Futuristic Streamer/Boxer Platform

## Positioning
Build a private concept platform that feels like a live fight-night command center, not a normal creator website.

**Core pitch:**
> This is the official headquarters concept for DeenTheGreat: live streams, fight archive, CrashoutBoyz merch, flyers, sponsors, socials, and upcoming events in one place.

## Visual direction
- Dark mode only.
- Black + white + acid neon green.
- Futuristic cyber combat + premium streetwear.
- Distressed sharp typography.
- HUD panels, scanlines, green glows, arena smoke, ring rope separators.
- Clean structure even with motion.

## Signature UI ideas
1. **Boxer Walkout Intro** — 5–7 seconds, first visit only, skip button, reduced-motion fallback.
2. **Ring Bell + Swinging Gloves Navbar** — top center, gloves swing on scroll/hover.
3. **Reactive Arena Background** — mouse/scroll reactive green glow, subtle grid/smoke.
4. **CrashoutBoyz as merch lane** — concept preview only until approved.

## Main site sections
- Home / Hero
- Live
- Fights
- Events
- CrashoutBoyz
- Flyers
- Socials
- Media
- Contact / Sponsors
