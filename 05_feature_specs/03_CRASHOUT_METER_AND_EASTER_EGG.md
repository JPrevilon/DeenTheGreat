# Crashout Meter + Easter Egg

## Crashout Meter
A sticky/floating meter that changes as the user scrolls.

States:
1. CALM
2. LOCKED IN
3. PRESSURE RISING
4. CRASHOUT MODE
5. BELL RINGS

## Easter Egg
If the user clicks the hanging gloves 5 times:
- Show “CRASHOUT MODE ENABLED”
- Add a short green glitch overlay
- Shake the page subtly
- Reveal a hidden COB teaser card
- Do not make the effect annoying or permanent
