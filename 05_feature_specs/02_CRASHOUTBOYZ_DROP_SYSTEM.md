# CrashoutBoyz Drop System

## Goal
Make CrashoutBoyz feel like a real merch brand, not a simple product grid.

## Must have
- Drop 001: Crashout Boyz
- Coming soon state
- Waitlist CTA
- Product cards: heavyweight tee, hoodie, fight-night tee, COB cap
- Stock/status labels: `Coming Soon`, `Limited`, `Concept Preview`
- Fine print: “Concept collection preview. Not available for purchase until approved.”

## Design
Use black/white/acid green, distressed edges, HUD cards, and merch-print-ready assets from `brand_pack_v3`.
