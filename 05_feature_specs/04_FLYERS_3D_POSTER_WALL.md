# Flyers 3D Poster Wall

## Goal
Turn the uploaded fight/stream flyers into a premium interactive wall.

## Behavior
- CSS 3D perspective grid.
- Hover zooms poster forward.
- Click opens a modal with larger poster and event details.
- Mobile fallback: clean swipeable grid.
- Use `/deen-assets/flyers/*`.
- Never crop important text too aggressively.

## Performance
- Use Next Image if possible.
- Lazy load below the fold.
