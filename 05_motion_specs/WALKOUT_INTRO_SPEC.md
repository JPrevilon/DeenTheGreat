# Walkout Intro Motion Spec

## Timing
Total: 5–7 seconds max.

## Sequence
1. Black screen with acid green text: ENTERING THE ARENA.
2. Forward tunnel movement using radial gradients, grid, crowd silhouettes, camera bounce.
3. White flash/photo moment.
4. Ring ropes slide upward into view.
5. Ring bell hit / acid green flash.
6. Homepage hero appears.

## Rules
- Add Skip Intro button.
- Store `dtg-intro-seen=true` in sessionStorage.
- Respect `prefers-reduced-motion`.
- Mobile intro should be simplified.
- Do not force audio. Add optional muted-by-default audio later.
